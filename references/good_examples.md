## Instructions for the script-writer skill

When this brand voice file is provided alongside a script-writing task:

1. **Read it before drafting.** Don't write a single line without this context.
2. **Apply tone calibration first.** Match the spectrum positions in section 5.
3. **Use the words from section 3.** Avoid the words from section 4.
4. **Mirror the sample writing in section 10.** If your draft doesn't sound like that sample, rewrite.
5. **Honor the "always" and "never" rules.** These supersede default skill behavior when they conflict.
6. **For each variation delivered, do a final voice check** against this file before showing the user.

If the brand voice in this file conflicts with the default skill voice (e.g., a luxury brand wants formal voice), brand voice wins — but specificity, "you" ratio, and the hook-type discipline still apply regardless of voice.
