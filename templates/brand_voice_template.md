# Audience Profile (ICP)

This is a fill-in template. Fill it out for each distinct audience you serve. Save the completed version. The script-writer skill will read this on every script task and target voice/hook selection to match.

> **How to use:** Fill in the sections below with specifics about ONE specific person — your ideal customer. Don't write for "an audience" — write for one specific human. If you serve multiple audiences, create one file per audience.

## Instructions for the script-writer skill

When this audience profile is provided alongside a script-writing task:

1. **Read it before drafting.** The skill writes to one specific person, not a generic audience.
2. **Use the temperature from section 12** to pick hook type (per hook selection matrix in `references/hooks.md`).
3. **Use the pain language from section 11** — those are the exact words to use in pain hooks.
4. **Use the self-description from section 10** — those are the exact words to call them.
5. **Address objections from section 7** in the body of longer scripts (especially VSLs and webinars).
6. **Use false beliefs from section 8** as the Three Secrets in any webinar script.
7. **Frame the desired state using section 6** — vivid, specific, their version of winning.
8. **Test against this profile** before delivery: does this script speak to THIS person? Not to a generic audience.

If multiple audience profiles exist, ask the user which one this script is targeting before writing.
